Java Calculators

During my time in class learning about Java, I made some general use calculators, such as checking if a number is even or odd shown here:

# Even or Odd in Java
class Main {

public static void main(String[] args) {

int numToExamine = 5;
int moduloOfNum;
moduloOfNum = numToExamine % 2;

System.out.println("The remainder of the given number, divided by two is: " + moduloOfNum);
System.out.println(moduloOfNum);
}
}

This code gets the modulo (remainder) of a integer which isn this case is 5, and the number 2. If the modulo of 5 and 2 is 1, then the number is odd. If the number is 0, then the number is even. Some pretty simple stuff.
