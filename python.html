Firstly I needed to get the current time and the desired time. I looked for the only major python library I would need for this project. It is called datetime. I used this library to get the current time and find the difference between desired time and the current time.I got the current time by using datetime.datetime.now() I quickly realized I needed to get a format of time to convert the current and the standard times to work together through the library. So I wrote a function to ask the user for their desired time, asking them to use the '%Y-%m-%d %H:%M:%S' format. I then combined their input with a format so the library could comprehendtheir answer as a time instead of a string of gobbledygook. Now all that was left was to make the actual timer. I found out how to do that with a minimal amountof research online. I got the formatted current time using this:
