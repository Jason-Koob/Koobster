microSIEM Project

I have worked with a System Information Event Management (SIEM) system before. I like how these systems can hold large amounts of information and use the stored information to do functions, with the information. I want to make something like a SIEM in JavaScript. I want to use a pre-made log of events, allowing users to add more. I want the site to be able to sort by columns and select information within the columns. I want to graph data, and automatically generate false logs to simulate actual data flowing in to the system from Event logging systems throughout the network.

I've worked with Malwarebytes and LogRhythm SIEM systems previously and enjoyed how they managed data.
