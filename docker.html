Docker Projects

I like working with docker. The containerization of instances makes sense logistically. I like spinning up a new docker container to try something I want to learn about. I like how easily variables can be changed through a docker command or inside a docker compose file which contains other environmental variables used by the container.

Some Of My Docker Containers

I have a few docker containers I use frequently. I have a docker container containing a Minecraft image. When I was starting to get used to docker and how it works, this was one of the first containers I used from the Docker hub. I liked how easy it was to pass in variables to the compose file and then see those same variables used in the container. Docker also makes it easy to read error logs when something does wrong which happens on ocassion.
